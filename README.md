# Library Management System

This is a simple library management app created to practice Python Object Oriented Programming, Streamlit, SQLite and software engineering principles.

## Features

- Python OOP backend
- SQLite relational database
- Streamlit UI

## How to run:

1. Create a virtual environment:

```
python -m venv venv
source venv/bin/activate # Mac/Linux
venv\Scripts\activate # Windows
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run Streamlit:

```
streamlit run app.py
```
